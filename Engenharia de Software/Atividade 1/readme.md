  - Postar Solução de Atividades/Tarefas
  - Enviar mensagem no Chat
 
- Já no segundo caso com o Professor, o ator também pode fazer 3 ações principais, como:
  - Criar uma Atividade/Tarefa
  - Enviar mensagem no Chat
